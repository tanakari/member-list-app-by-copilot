# 仕様概要・一覧

## 概要

メンバー情報を管理するシンプルなWebアプリケーション。メンバーの登録・更新・削除・検索・一覧表示を行う。

## 画面一覧

| No  | 画面名             | 概要                                   | 仕様書                           |
| --- | ------------------ | -------------------------------------- | -------------------------------- |
| 1   | メンバー登録画面   | メンバー情報を登録する画面             | specs/ui/member-registration.md |
| 2   | メンバー一覧画面 | 登録したメンバー情報を一覧表示・検索する画面 | specs/ui/member-list.md         |

## 非機能要件

- レスポンスは即時（1秒以内）
- UIはスマホ・PC両対応（レスポンシブ）
- セキュリティ：XSS等の基本的なWeb対策
